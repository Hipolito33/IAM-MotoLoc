# Documento de Requisitos

## Objetivo

Desenvolver um sistema de gerenciamento interno para uma empresa de aluguel de motocicletas.

---

# Cadastro de Clientes

O sistema deverá permitir:

* cadastrar clientes
* editar clientes
* excluir clientes
* listar clientes
* pesquisar clientes

Informações:

* Nome
* CPF
* RG
* CNH
* Validade da CNH
* Telefone
* E-mail
* Endereço

---

# Cadastro de Motos

O sistema deverá permitir:

* cadastrar motos
* editar motos
* excluir motos
* listar motos

Informações:

* Placa
* Modelo
* Marca
* Ano
* Cor
* Quilometragem
* Valor da diária
* Status
* Observações

---

# Cadastro de Contratos

O sistema deverá permitir:

* criar contratos
* editar contratos
* finalizar contratos
* cancelar contratos

Informações:

* Cliente
* Moto
* Data de início
* Data prevista de devolução
* Valor da diária
* Valor do caução
* Observações
* Status

---

# Documentos

Cada contrato poderá possuir:

* Contrato PDF
* CNH
* Comprovantes
* Fotos da entrega
* Fotos da devolução

---

# Dashboard

Exibir:

* motos disponíveis
* motos alugadas
* motos em manutenção
* contratos ativos
* contratos vencendo
* receita
